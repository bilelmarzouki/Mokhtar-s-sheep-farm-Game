# Mokhtar's Farm Game

A browser-based game built with vanilla JavaScript, HTML, and Css (developed in VS Code).
Your mission is to catch wandering sheep and deliver them safely into the barn before time ends.
![image alt](https://github.com/bilelmarzouki/Mokhtar-s-sheep-farm-Game/blob/5aa015ff9e3415f00b13e7795eca1f89a2750123/start_Game.PNG)
## How to Play

1) Click **Start** to begin the game.
2) Move the farmer using the arrow keys (Up,Down, Left,Right).
3) Collide with a sheep to catch it.
4) Carry the sheep to the barn(left-side-gate).The gate opens when you get close.
5) Press **space** to release the sheep inside the barn.
6) Save more thatn **50%** of the sheep to wiwn; otherwise ,you lose.
7) Clicck **Restart** to restart the game.

## Controls
-Arrow Keys: Move the farmer (Up / Down / Left / Right)
-Space: Release sheep into the barn.
-Start Button: Start the game.
-Restart Button: Restart the game.

## Game Features
1) Start / Restart button: Start a new game or restart anytime.
2) Random sheep movement: Sheep move randomly around the play area.
3) Keyboard-controlled farmer: The farmer is controlled via the arrow keys.
4) Timer (120 seconds): You have 120 seconds to save as many sheep as possible.
5) Farmer ↔ Sheep collision (catching): Colliding with a sheep lets the farmer catch/carry it.
6) Sheep ↔ Barn rule: Sheep can’t enter the barn by themselves—only when the farmer brings them in.
7) Score tracking (Saved): The saved score updates after each sheep is released inside the barn.
8) Smart barn gate: The left gate opens when the farmer is close and closes when he leaves.

## Win / Lose Conditions

- Win: Save more than 50% of the sheep before time runs out.
- Lose: Save 50% or less when the timer reaches 0.

## How to Run (VS Code)

1) Open the project folder in VS Code.
2) Install the “Live Server” extension.
3) Right-click `index.html` → **Open with Live Server** .
